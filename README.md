# 💻 Fuzzy Mamdani Laptop Recommendation System

This project is a Decision Support System (DSS) developed for the Sistem Cerdas Pendukung Keputusan (SCPK) course.  
The application uses the Fuzzy Mamdani method to recommend the best laptop based on user preferences such as price, RAM, and storage capacity.

The system performs:
- Fuzzification
- Rule inference
- Aggregation
- Defuzzification
- Laptop ranking process

Built with an interactive web interface using Streamlit.

---

## 🚀 Features

- Fuzzy Mamdani implementation
- Laptop recommendation ranking
- Interactive dashboard
- Membership function visualization
- Dataset visualization
- Multiple defuzzification methods
